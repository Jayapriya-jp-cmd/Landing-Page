# Landing Page - Internship Submission

## 🔥 Project Overview

This is a responsive blog website built using **HTML5** and **Tailwind CSS**. It features:

- Responsive Navbar
- Hero and Services sections
- Pricing cards with hover effects
- Stories and Footer section

## 🛠️ Tech Stack

- HTML
- Tailwind CSS
- Google Fonts
- Icons (Flaticon)
